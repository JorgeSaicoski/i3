# i3 to Fedora (or DNF package)
### i3 config

## Run!
```bash
. ./run.sh
```

## Comands
### super + p = Dolphin
### super + m = System Monitor
### super + b = Brave (browser)
### super + shift + m = spotify
### Print = open screenshot software
### super + Print = screenshot

## Multiple Monitor
```bash
sudo dnf install xrandr
xrandr
```
